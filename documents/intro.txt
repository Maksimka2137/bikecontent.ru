Тема проекта: «сайт-лендинг для продвижения Youtube-канала Road-Eagles»

Актуальность проекта: сейчас многие люди стараются вести активный образ жизни, например любят кататься на велосипеде.
Помимо занятий спортом, часто хочется показать и рассказать окружающим о своём образе жизни.
Популярный способ сделать это — создать канал на Youtube (или RuTube) и выкладывать туда ролики про то, что тебе нравится.
Для того, чтобы привлечь внимание к каналу, мы решили сделать web-сайт.

Цель проекта: научиться разработке web-сайтов, научиться работать с большим массивом информации, привлечь как можно больше людей к нашему Youtube-каналу.

Поставленные задачи для достижения цели:
    1. Найти и пройти обучающие курсы по основам web-технологий
    2. С помощью курсов получить недостающие знания для создания сайта
    3. Зарегистрировать доменное имя 
    4. Изучить дополнительную информацию, которая понадобится для выполнения проекта
    5. Подобрать шаблон для оформления сайта
    6. Подготовить контент (материалы для наполнения сайта — текст, изображения, видеоролики) 
    7. Создать сайт и наполнить его информацией

Этапы работы над проектом:
    1. Выбор темы (интерес, возможность, польза и актуальность)
    2. Подготовка документации и сбор начальных данных (заполнение «брифа» на создание сайта, создание структуры, запись на обучающие курсы для получения недостающих знаний (The Odin Project))
    3. Подготовка материалов (написание текста на русском и английском языках, подбор иллюстраций и видеороликов)
    4. Выбор вариантов оформления будущего сайта
    5. Подготовка технической площадки (создание репозитория для хранения исходного кода, подготовка и аренды VPS сервера, аренда доменного имени, настройка сервера, настройка процесса развёртывания проекта)
    6. Наполнение шаблона контентом
    7. Наполнение англоязычной версии сайта
    8. Реализация формы обратной связи на сайте (чтобы была возможность отправить сообщение с сайта в Telegram)
    9. Создание группы в Telegram для приёма заявок (группа в Tg, бот для отправки сообщений, подключение к сайту)
    10. Подключение системы аналитики (Yandex Metrica)
    11. Проверка работоспособности сайта
    12. Составление документации по проекту

Методы и приёмы исследования и работы над проектом:
    - Проведение тестовых испытаний сайта
    - Анализ необходимых теоретических источников
    - Выявление недочётов в работе сайта
    - Устранение этих недочётов
    - Размещение сайта в сети Интернет
